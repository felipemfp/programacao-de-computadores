## Expressões Matemáticas em Ruby

a) 1 + 5 - 2

```ruby
1+5-2
```

b) 4 / 2 x 5

```ruby
4/2*5
```
c) 1,2 + 2,4

```ruby
1.2+2.4
```

d) 3,6 / 3 + 2,5

```ruby
3.6/3+2.5
```

e) 6 + 4 / 2 + 3 (Fração)

```ruby
(6+4)/(2+3)
```

f) 5 + 3 x 10

```ruby
5+3*10
```

g) 25% de 40

```ruby
40*0.25
```

h) 4 x [3 - 4 x (7 - 2) + 1]

```ruby
4*(3-4*(7-2)+1)
```

i) 1 / 5

```ruby
1/5
```

j) 1,5 / 3

```ruby
1.5/3
```
